### q026 수동 검토

자동 평가에서는 네 검색기 모두 `q026`을 Hit@5 실패로 계산했습니다. 골든셋은
`모집기간 : 공고일 ~ 26. 2. 24.(화) 18:00까지`가 들어 있는 Chunk만 relevant로
표시했기 때문입니다.

하지만 BM25 1위 결과에는 다음과 같은 별도 유효 근거가 있었습니다.

```text
제출기간 : 26. 2. 24.(화) 18:00
```

즉 실제 답변은 가능하지만 단일 answer span 기반 자동 판정이 놓친 false negative입니다.
포트폴리오 수치는 사후에 올리지 않고 Hit@5 0.90을 그대로 유지하며, 자동 metric과
사람 검토를 함께 써야 하는 사례로 기록합니다.

## 읽는 방법

- Hit@k: 상위 k개 안에 정답 Chunk가 들어간 질문의 비율
- MRR: 첫 정답이 위에 있을수록 커지는 평균 역순위
- nDCG@k: 여러 정답 Chunk가 있을 때도 위쪽 배치를 더 높게 평가
- 평균/p95 ms: 품질과 함께 보는 응답 지연 참고값

dev 결과로 설정을 고르고, test split은 최종 선택 뒤 한 번 확인합니다.
