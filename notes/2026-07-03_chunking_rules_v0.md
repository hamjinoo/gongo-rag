# Chunking Rules v0

## 목표

공고문을 질문에 답할 수 있는 근거 단위로 나눈다.

## 기본 규칙

1. `Ⅰ, Ⅱ, Ⅲ, Ⅳ, Ⅴ`는 큰 섹션 제목으로 기억한다.
2. `3. 모집 절차`, `4. 참여 가능 기관` 같은 줄은 chunk 시작점 후보로 본다.
3. 상위 제목 아래에서 다시 `1, 2, 3...`으로 시작하는 항목은 목록 item으로 보고, 직전 제목에 포함한다.
4. `ㅇ`, `-`, `※`는 단독 chunk로 만들지 않고 직전 제목에 붙인다.
5. 다음 같은 레벨 제목이 나오면 현재 chunk를 종료한다.
6. chunk에는 항상 상위 제목 path를 붙인다.
