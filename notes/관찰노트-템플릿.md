## 관찰 체크리스트 (docs/text/의 .txt를 열고 원본 PDF와 비교하며)

- [ ] **표**는 어떤 모양으로 추출됐나? (줄이 깨졌나, 셀 순서가 섞였나, 아예 사라졌나)
  - 예시 붙여넣기:
- [ ] **줄바꿈**이 문장 중간에서 끊기는 곳이 있나? (PDF의 시각적 줄바꿈이 텍스트 줄바꿈으로)
- [ ] **머리말/꼬리말**(페이지 번호, 기관명)이 본문 중간중간 반복되나?
- [ ] **목록/번호 체계**(1. 가. 1) □ ○)는 살아 있나?
- [ ] **문단 구분**(빈 줄)이 남아 있나? → 문단 기반 chunking이 가능할지의 힌트
- [ ] 자격/금액/기한 같은 **핵심 정보**는 어디에 있나? (본문 문장 vs 표 안)
- [ ] 이상한 특수문자, 깨진 글자가 있나?

## 이 관찰이 chunking에 주는 시사점 (스스로 써보기)

- 고정 크기로 자르면 무엇이 걱정되나?
- 문단 기반으로 자르면 무엇이 걱정되나?
- 정답이 표 안에 있는 질문은 어떤 운명일 것 같나? (→ 8주차 실패 분석의 씨앗)
